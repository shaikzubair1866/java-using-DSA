Control Statements in Java – Detailed Notes
Definition

Control statements are used to control the flow of execution of a program. They decide which statements execute, how many times they execute, and when execution stops.

Types of Control Statements
1. Selection (Decision-Making) Statements

These statements help the program make decisions based on conditions.

a) if Statement

Purpose: Executes a block of code only when the condition is true.

Syntax
if(condition){
    // statements
}
Example
int age = 18;

if(age >= 18){
    System.out.println("Eligible to vote");
}
Explanation
Condition age >= 18 is checked.
If true, the statement inside if executes.
If false, nothing happens.
b) if-else Statement

Purpose: Executes one block if the condition is true and another block if it is false.

Syntax
if(condition){
    // true block
}
else{
    // false block
}
Example
int num = 5;

if(num % 2 == 0){
    System.out.println("Even");
}
else{
    System.out.println("Odd");
}
Explanation
If the number is divisible by 2, print "Even".
Otherwise, print "Odd".
c) else-if Ladder

Purpose: Used when multiple conditions need to be checked.

Syntax
if(condition1){
}
else if(condition2){
}
else{
}
Example
int marks = 85;

if(marks >= 90){
    System.out.println("Grade A");
}
else if(marks >= 75){
    System.out.println("Grade B");
}
else{
    System.out.println("Grade C");
}
Explanation
Conditions are checked one by one.
The first true condition executes.
Remaining conditions are skipped.
d) switch Statement

Purpose: Selects one block of code from multiple choices.

Syntax
switch(expression){
    case value1:
        statements;
        break;

    case value2:
        statements;
        break;

    default:
        statements;
}
Example
int day = 2;

switch(day){
    case 1:
        System.out.println("Monday");
        break;

    case 2:
        System.out.println("Tuesday");
        break;

    default:
        System.out.println("Invalid Day");
}
Explanation
The value of day is compared with each case.
Matching case executes.
break stops further execution.
default executes when no case matches.
2. Iteration (Looping) Statements

Loops are used to execute a block of code repeatedly.

a) for Loop

Purpose: Used when the number of repetitions is known.

Syntax
for(initialization; condition; increment){
    // statements
}
Example
for(int i = 1; i <= 5; i++){
    System.out.println(i);
}
Explanation
Initialize i = 1
Check condition i <= 5
Execute statements
Increment i
Repeat until condition becomes false
Output
1
2
3
4
5
b) while Loop

Purpose: Executes as long as the condition is true.

Syntax
while(condition){
    // statements
}
Example
int i = 1;

while(i <= 5){
    System.out.println(i);
    i++;
}
Explanation
Condition is checked before execution.
Loop stops when condition becomes false.
c) do-while Loop

Purpose: Executes the block at least once.

Syntax
do{
    // statements
}while(condition);
Example
int i = 1;

do{
    System.out.println(i);
    i++;
}while(i <= 5);
Explanation
Statements execute first.
Condition is checked later.
Executes at least one time even if condition is false.
3. Jump Statements

Jump statements transfer control from one part of the program to another.

a) break Statement

Purpose: Terminates a loop or switch statement immediately.

Example
for(int i = 1; i <= 5; i++){
    if(i == 3)
        break;

    System.out.println(i);
}
Output
1
2
Explanation
When i becomes 3, break stops the loop.
b) continue Statement

Purpose: Skips the current iteration and moves to the next iteration.

Example
for(int i = 1; i <= 5; i++){
    if(i == 3)
        continue;

    System.out.println(i);
}
Output
1
2
4
5
Explanation
When i is 3, that iteration is skipped.
c) return Statement

Purpose: Exits from a method and optionally returns a value.

Example
public static int add(){
    return 10;
}
Explanation
return ends the method execution.
Returns the value 10.
Summary Table
Type	Statements	Purpose
Selection	if, if-else, else-if, switch	Decision making
Iteration	for, while, do-while	Repeated execution
Jump	break, continue, return	Transfer control
